# MansaPay

Blockchain remittance platform for affordable cross-border money transfers
in West Africa. Pilot corridor: Senegal <-> The Gambia, settled on the
Stellar network using USDC.

## Mission
Use blockchain technology to widen financial inclusion and create economic
opportunity for people across Africa.

## Project Status
Pre-alpha. Currently in Sprint 0 (scaffolding).

## Tech Stack
- Backend: Node.js 20 + Express + TypeScript
- Database: PostgreSQL 15, Redis 7
- Blockchain: Stellar SDK, USDC
- Web: React 18 + Vite
- Mobile: React Native (Expo)
- Infrastructure: Docker Compose (local), AWS EKS (production)
